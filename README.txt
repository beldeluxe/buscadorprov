La aplicación consiste en un buscador de provincias.
Funciona mediante un input que se autompleta,
una vez seleccionada la provincia mostrará un mapa con ciudades/municipios (cinco) que se encuentran en la provincia con sus marcadores.
La aplicación funciona con una base de datos sql, archivo que adjunto con el nombre de "buscaprovincias.sql", por tanto habrá que cargar la base de datos previamente para que la aplicación funcione correctamente.

He usado jquery y php con Ajax, para que funcione tendrá que ejecutarse desde servidor.
